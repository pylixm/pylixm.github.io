yum install Header V3 RSA/SHA256 Signature, key ID 0608b895

约 220 字小于 1 分钟

yum

2023-09-08

Q & A：

使用 yum 安装软件是出现以下错误：

	Total size: 8.3 M
	Is this ok [y/N]: y
	Downloading Packages:
	warning: rpmts_HdrFromFdno: Header V3 RSA/SHA256 Signature, key ID 0608b895: NOKEY
	Public key for openpgm-5.1.118-3.el6.x86_64.rpm is not installed

原因，这是由于yum安装了旧版本的GPG key或者key文件不存在s造成的

解决方案：

第一种方式： 使用如下命令导入

	rpm --import /etc/pki/rpm-gpg/RPM*

如果这种方式不行，用下面的方式：

	vim /etc/pki/rpm-gpg/RPM-GPG-KEY-EPEL

使用上门的命令打开文件，并把下面的代码粘贴到文件中。
